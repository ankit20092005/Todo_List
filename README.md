# 📝 Todo List App

A simple and responsive Todo List application built with React and Tailwind CSS to manage daily tasks efficiently.

## 🚀 Features
- Add and delete tasks
- Clean and responsive UI
- Fast performance with Vite

## 🛠️ Tech Stack
- React.js
- Tailwind CSS
- Vite

## ⚙️ Setup

```bash
git clone https://github.com/YOUR_USERNAME/todo-list.git
cd todo-list
npm install
npm run dev
